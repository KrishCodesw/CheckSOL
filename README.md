# CheckSOL
CheckSOL is a decentralized web application built on the Solana blockchain, offering users a seamless experience for managing Solana-based wallets, interacting with the blockchain, and participating in token airdrops. The app provides a set of core features, ensuring both ease of use and functionality



Key Features:
Solana Wallet Integration:

Connect to Solana wallets through the Solana Wallet Adapter.
Supports wallet actions like connect and disconnect.
Real-time wallet balance tracking.
Airdrop Functionality:

Users can request airdrops directly within the app, using the Solana faucet (on devnet/testnet).
Provides airdrop history to track previous airdrops with details like address, amount, and date.
Token Launchpad:

Allows users to interact with and participate in new token launches within the Solana ecosystem.
User-friendly Interface:

A responsive and clean UI built with Tailwind CSS to ensure a smooth experience across devices.
Includes a hamburger menu for small screen sizes, ensuring access to important features like wallet actions and social links.
Social Media Integration:

Instagram, GitHub, Twitter, and LinkedIn links are embedded for users to connect and stay updated on social platforms.
Design:
The app features a modern dark theme with a gradient logo, clean typography, and intuitive navigation.
Mobile responsiveness: The UI adapts to smaller screen sizes, ensuring accessibility on both desktop and mobile devices.
Target Audience:
CheckSOL is designed for developers, enthusiasts, and anyone interested in interacting with the Solana blockchain, from beginner users seeking wallet integration to advanced users participating in token airdrops.
